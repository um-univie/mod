int getGraphIndex() const¶
Returns:

The index of the graph in the owning Union graph this vertex is from. The index is thus in the range \([0, len(getGraph())[\).

We can obtain the underlying Graph this vertex is from both directly via getVertex() as getVertex().getGraph(), or via the graph index as getGraph()[getGraphIndex()].
